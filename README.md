# NCL Lunabotics Challenge

This repository contains the software for the Newcastle University Lunabotics robot.

## Structure: 
- 'ros2_ws/' : ROS 2 workspace
    - Contains all ROS packages (drivers, control, autonomy, etc)

## Requirements
- Ubuntu 22.04
- ROS2 Humble
- colcon

## Build Instructions
``bash
cd ros2_ws
colcon build
source install/setup.bash

## Gazebo Simuatlions
